Pour le Bon fonctionnement de l'application T'chat veuillez suivre les indications suivantes:

1 - Déplacer le dossier de l'application télecharger dans le dossier de votre serveur local pour mon cas c'est C:/xampp/htdocs
2 - Le dossier parent de l'application doit obligatoirement avoir comme nom "MVC_project". (Important)
3 - Importer la base de donnée dans votre phpmyadmin à l'aide du fichier existant dans MVC_project/BDD/chat_v1.sql
4 - Pensez à changer le nom d'utilisateur et le mot de passe phpmyadmin pour donner l'accés à votre BD. Vous pouvez le faire
    directement sur le fichier MVC_project/model/Manager.php

Cordialement!!!
